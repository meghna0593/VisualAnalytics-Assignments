******Meghna R Holla B00812604******

Code to implement the basic functionality of Radviz Chart using D3.js library.

Features Added:
1. Drop Down with different CSV file selection. 
2. Correlation Matrix on hovering over the data points.
3. Enter the number of clusters and click on 'Create Cluster' button to create different clusters using K-Means algorithm. 
4. On clicking 'Create Cluster' without any values will create 5 clusters by default.
5. Correlation Matrix is displayed on hovering over data points grouped into clusters as well
6. On clicking 'RadViz with Class' one can switch back to radviz with classes from clusters.
7. On selecting a different file using the drop down menu, one can go back to the default chart by clicking 'Default Radviz'
8. Click on 'Bonus' to view the bonus functionality. Select the categorical columns and click on 'Create Radviz Plot' to create a plot using Assignment 1 processed data.

Features from Assignment 2:
1. Default Radviz Chart on page load 
2. Check boxes to make Anchor selection. The chart re-renders based on checked/unchecked checkboxes.
3. Slider to change the opacity of the data points
4. Anchors can be moved around on the circumference of the circle. 
5. Legend is displayed. On hovering over the values on the legend, respective points will get highlighted
6. On hovering over each data point, details wrt to that point is shown on tooltip
7. On hovering over each anchor, tooltip to determine the position of anchors is given.
8. On moving the anchors on the circle, one can go back to the default anchor positions by clicking 'Reset Anchor Position'
9. The data used are normalized
